Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined in the beginning element, never the closing element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the conent inside an element.
  • [link] creats a connection to an external file
  • A selector defines the element or attribute to which the rules or declarations will apply.
  • Declarations contain the CSS property we want to apply and the value of the property.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git branch: lists your branches.
  • git merge [branch]: merge the specified branch's history into the current one.
